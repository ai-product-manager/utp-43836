# utp-43836

Landing page para un concesionario que vende Volkswagen y Audi.

## ¿Qué contiene?
- `index.html` — Página principal con hero, secciones de modelos, ofertas, financiación y formulario de contacto.
- `styles.css` — Estilos responsivos para la página (header, hero, cards, grid, formulario).

## Cómo probar
1. Abre `index.html` en tu navegador (doble clic sobre el archivo o arrástralo a una ventana del navegador).
2. En móvil, prueba el botón del menú superior para ver la navegación con el diseño responsivo.
3. El formulario es un ejemplo con validación básica y muestra una alerta tras enviar (no envía correos reales).

## Notas
- Las imágenes usan enlaces de Unsplash (públicas y gratuitas). Reemplaza por imágenes propias si lo deseas.
- Para convertir el formulario en algo funcional en producción: conecta a un back-end o un servicio de email (ej. Formspree o una API propia).
